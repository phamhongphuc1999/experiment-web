# Experiment Web

A personal experimental project showcasing various web technologies, interactive games, and utilities.

## Features

- **Caro (Tic-Tac-Toe)**: Multiple modes including offline, online multiplayer, and blind mode.
- **Connect4**: Classic disc-dropping game.
- **Pikachu**: Tile-matching game with various board transformations.
- **Web3 Wallet**: Integration and interaction with Web3 wallets.
- **Crypt**: Secure encryption/decryption utility.

## Getting Started

### Prerequisites

- [Bun](https://bun.sh/) installed on your machine.

### Installation

```shell
bun install
```

### Development

```shell
bun dev
```

## Deployment

## Tech Stack

- **Framework**: Next.js 15
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **State Management**: Zustand
- **Animations**: Motion (framer-motion)
